# Minha Página Web

Uma página web moderna e responsiva criada do zero com HTML, CSS e JavaScript.

## 📋 Conteúdo

- **index.html** - Estrutura HTML da página
- **styles.css** - Estilos CSS responsivos
- **script.js** - Funcionalidades JavaScript

## 🎨 Características

✅ Design responsivo (funciona em desktop, tablet e mobile)
✅ Navegação com scroll suave
✅ Seções bem organizadas (Hero, Sobre, Serviços, Contato)
✅ Formulário de contato interativo
✅ Animações e transições suaves
✅ Código limpo e bem comentado

## 📱 Seções

1. **Navegação** - Menu sticky com links para as seções
2. **Hero** - Seção inicial com chamada para ação
3. **Sobre** - Informações sobre o site
4. **Serviços** - Grid com 3 serviços principais
5. **Contato** - Formulário para entrar em contato
6. **Footer** - Rodapé com informações

## 🚀 Como Usar

1. Abra o arquivo `index.html` em seu navegador
2. Explore as diferentes seções clicando nos links de navegação
3. Teste o formulário de contato
4. Personalize o conteúdo conforme necessário

## 💡 Como Personalizar

### Mudar o título da página
Edite em `index.html`:
```html
<title>Seu Novo Título</title>
```

### Mudar cores
Edite em `styles.css`:
- Cor primária: `#3498db`
- Cor secundária: `#2c3e50`
- Gradiente do hero: `#667eea` e `#764ba2`

### Adicionar mais seções
Copie uma seção existente e adicione seu próprio conteúdo.
